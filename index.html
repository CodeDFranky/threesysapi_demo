What is this demo?

What we have here is a demo application making use of our developed API. Our API functions in either one of two ways: “/generate” or “/verify.”

Sign a document (using /generate)

When the app uses this endpoint, the API requires 2 things:

- a PDF document
- and a location of where you would like our novel, Stenographic Data Matrix, to be placed on. You have 4 options: top-left, top-right, bottom-left and bottom-right

The API will work to determine whether or not your document possesses the traits we look for when placing our signature.
If it does, then it will place a signature on the document and return it to you; Otherwise, you will be prompted as what could be the cause of the issue.

Upload a signed document (using /verify)

When the app uses this endpoint, the API requires only 1 thing:

- a PDF document

The API will work to determine whether or not your document is one that is valid, meaning it has been signed with our API, or otherwise.
